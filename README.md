# Gym Booking Bot

A Python-based gym booking system with Telegram / WhatsApp integration, reminders, attendance workers, and admin dashboard.

## Features
- Booking logic
- Telegram bot
- WhatsApp webhook
- Reminder workers
- Attendance workers
- Admin dashboard

## Setup
1. Create virtual environment
2. Install dependencies
3. Add `.env` file
4. Run the required scripts

## Notes
- `.env` is not included
- Database files are ignored
